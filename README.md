BloodFinder

BloodFinder is a smart, real-time platform that connects verified blood donors with patients in emergencies, saving critical time during blood shortages. Built for a hackathon by a team of BTech CS students, it solves India's blood crisis where 12,000+ people die daily due to slow manual donor searches via social media.

The Problem
- Chaotic manual searches: Families rely on social media posts and phone calls, causing fatal delays.
- o central database: Impossible to filter donors by blood group, location, or availability.
- Every minute counts: Current systems lack speed, precision, and verified donor data.

Our Solution
A centralized mobile/web app with:
- OTP-verified donor registry – Real profiles, no fakes.
- verification via digi locker
- GPS-powered matching – Finds nearest compatible donors instantly.
- Push notification alerts – 3x faster donor responses.(time sensitive)
- Secure & private – Encrypted data, minimal info shared until match confirmed.

Key Wins:
- 80% reduction in search time
- 100% verified donors only
- Real-time availability tracking

How It Works
```
Donor: Register → Verification → Set availability → Get matched
Recipient: Search blood group/location → Send request → Get donor match → Coordinate
```

Tech Stack (Planned)

- Backend: Node.js / Firebase

- Auth/Alerts: Twilio / Firebase Cloud Messaging
