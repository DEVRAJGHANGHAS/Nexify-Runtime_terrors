BloodFinder
Smart real-time blood donor matching app that slashes search times from hours to minutes, tackling India's daily 12,000+ blood shortage deaths caused by chaotic social media pleas.

The Problem
Families waste hours on unverified social media posts and calls.

No central database for blood group, location, or availability filters.

Delays kill: Every minute without blood is fatal.

Our Solution
Mobile/web app with:

OTP + DigiLocker verified donors (100% real profiles).

GPS matching for nearest compatible donors.

Instant push alerts (3x faster responses).

Encrypted privacy until match confirmed.

Key Wins:

80% faster searches.

Real-time availability.

Fully verified users.

How It Works
text
Donor: Register → Verify → Set availability → Get matched & alerted.
Recipient: Search by group/location → Request → Instant donor matches → Coordinate.
Tech Stack
Frontend: React Native / Flutter

Backend: Node.js / Firebase

Maps: Google Maps API

Auth/Alerts: Twilio / Firebase Cloud Messaging

DB: MongoDB / Firestore
